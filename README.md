# 💱 Currency Converter Application

A professionally designed and responsive **Currency Converter Web Application** built using **HTML5, CSS3, JavaScript, Node.js, and Express.js**.  
This project provides real-time currency conversion using live exchange rate APIs with a clean, modern, and user-friendly interface.


## 🚀 Features
- 🌍 Real-time currency conversion
- 💹 Live exchange rate API integration
- 🔄 Multiple international currency support
- ⚡ Fast and accurate calculations
- 📱 Responsive and modern UI design
- 🎯 Easy-to-use interface
- 🔐 Reliable API-based data fetching


## 🛠️ Technologies Used
### Frontend
- HTML5
- CSS3
- JavaScript


## 📌 Project Objectives
- To provide accurate and real-time currency conversion.
- To simplify international currency calculations.
- To create a clean and responsive user experience.


## 👨‍💻 Target Users
- Travelers
- Students
- International business professionals
- Financial enthusiasts
